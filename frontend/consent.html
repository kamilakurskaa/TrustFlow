Для анализа кредитной истории необходимо ваше согласие

Важная информация

Сервис TrustFlow запрашивает доступ к данным вашей кредитной истории через портал Госуслуги для проведения анализа.

Какие данные будут обрабатываться:

  • Паспортные данные
  • История кредитных обязательств
  • Текущие задолженности
  • Кредитные рейтинги
  • Платежная дисциплина

Безопасность данных

Шифрование

Все данные шифруются по стандарту AES-256

Блокчейн

Хранение в распределенном реестре

Анонимизация

Персональные данные заменяются хешами
